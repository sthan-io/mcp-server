# sthan-cli

Monorepo for [sthan.io](https://sthan.io) developer tools.

## Packages

| Package | Description |
|---------|-------------|
| [@sthan/core](packages/core) | TypeScript client for sthan.io APIs |
| [@sthan/mcp-server](packages/mcp-server) | MCP server for Claude Code, Cursor, VS Code |

## Development

```bash
npm install
npm run build --workspaces
```
